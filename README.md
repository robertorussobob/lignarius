# Lignarius
_Lignarius_: a _Docker Compose_ integrated environment, with many services.

Services available:

- Allure Framework.
- Kafka;
- Jenkins;
- Localstack;
- Memcached;
- MySQL;
- NGINX;
- OpenSearch;
- PostgreSQL;
- Redis;
- Wordpress;

_Upcoming_:

- ELK/EFK (Elasticsearch, Logstash/Fluentd, Kibana);
- Gitea.
- Grafana;
- MongoDB;
- Nextcloud;
- Prometheus;
